    📎 付録　AIスタック解説ガイド

    東光印刷 萬財塾資料「AIスタック学習マップ 別紙」より。各ツールの用途・導入方法・料金・アウトプット例をまとめています。

    🌐 Webアプリ開発系
    01. GitHub コード管理
    コードを保存・共有・チームで管理するプラットフォーム。「コードのDropbox」。NetlifyやVercelへの自動デプロイの起点となる。
    無料（privateリポジトリも無制限） GitHub Copilot 月$10〜
    東光印刷での活用例：社内ツールのバージョン管理 / Netlify・Vercelへの自動デプロイ基盤
    02. Netlify ホスティング
    GitHubと連携してWebサイトを自動公開するホスティングサービス。pushするだけで自動更新。静的サイト・Reactアプリの公開に最適。
    無料（月100GBまで） Pro 月$19〜
    東光印刷での活用例：TOKOシリーズサイトの公開・更新 / 社内向けWebツールの無料ホスティング
    03. Vercel ホスティング
    Next.jsと相性抜群の高性能ホスティングサービス。SSR・APIルートもそのまま動く。Next.jsアプリにはNetlifyよりVercelが向いている。
    無料（個人・非商用） Pro 月$20（商用利用）
    東光印刷での活用例：広島子育てナビなどNext.jsアプリの公開 / SSRによるSEO対応サイト
    04. Supabase データベース
    クラウド上のPostgreSQLをGUIで簡単に操作できるサービス。認証機能も内蔵。Book MemoriesもSupabaseで動いている。RLS設定を忘れずに。
    無料（500MB・5万行まで） Pro 月$25（8GB）
    東光印刷での活用例：Book Memoriesの読書データ管理 / 社員が入力→リアルタイム共有できるアプリ
    05. Firebase データベース／認証
    GoogleのアプリバックエンドサービスでFirestore（NoSQL DB）・認証・ホスティングがセット。Googleアカウント認証との連携が強力。販促計画アプリで使用中。
    無料（Spark：1GB・5万read/日） Blaze（従量課金）
    東光印刷での活用例：販促計画アプリのデータ共有 / Googleアカウントでのログイン機能
    06. Next.js / React フレームワーク
    現代のWebアプリ開発の標準フレームワーク。Claudeに「Next.jsで〇〇アプリを作って」と指示すれば自動生成。TOKOシリーズはすべてこれで構築。
    完全無料（オープンソース）
    東光印刷での活用例：TOKOシリーズ全サイト / TOKO School Navigator / 広島子育てナビ
    07. PWA スマホアプリ化
    Progressive Web App。App Store申請不要でWebサイトをホーム画面に追加できる仕組み。manifest.json＋Service Worker＋アイコン2サイズで実現。
    完全無料（技術仕様）
    東光印刷での活用例：店舗視察レポートのスマホアプリ化 / 週刊AIノートのホーム画面追加
    🧠 プロンプト設計系
    08. RAG AI連携
    Retrieval-Augmented Generation。社内文書をデータベース化し、AIが質問に答える際に関連文書を自動参照する仕組み。「社内専用AI」の核となる技術。
    構築コストのみ（API従量課金） Claude API：100万トークン$3〜
    東光印刷での活用例：20年分の販促ノウハウを検索できる社内AI / 過去提案書から類似事例を自動提案
    ⚙️ 自動化・連携系
    09. Make（旧Integromat） ノーコード自動化
    ノーコードで複数ツールを自動連携するワークフロー自動化ツール。「Gmailに届いたら→Slack通知→スプレッドシートに記録」のような処理をGUIで設計できる。
    無料（月1,000オペレーション） Core 月$9〜（月1万オペレーション）
    東光印刷での活用例：問い合わせフォーム→Slack通知 / SNS投稿の定期自動化 / 受注メール→スプレッドシート自動記録
    10. GAS（Google Apps Script） 自動化スクリプト
    Gmail・スプレッドシート・Google Driveなどを自動操作できるGoogleの無料スクリプト環境。JavaScriptベースだがClaudeに頼めば自動生成。時間トリガーで定期実行も可能。
    完全無料
    東光印刷での活用例：日報の自動集計・メール送信 / スプレッドシートのデータ自動整理 / Googleフォーム回答の自動振り分け
    📊 データ分析系
    11. Python（pandas） データ分析
    Pythonはシンプルで読みやすいプログラミング言語。pandasはExcelのような表データを高速処理できるライブラリ。Claudeに「このCSVを分析して」と頼めばコードを生成してくれる。
    完全無料（オープンソース）
    東光印刷での活用例：残業・勤怠データの分析レポート / 売上データの集計・グラフ化 / 印刷コスト最適化の数値分析
    12. Looker Studio BI・可視化
    旧Google Data Studio。スプレッドシート・BigQuery・CSVなどのデータをグラフ・表・ダッシュボードとして可視化できるGoogleの無料ツール。更新すると自動反映。共有URLで社内共有も簡単。
    完全無料
    東光印刷での活用例：経営会議用の売上ダッシュボード / 顧客別受注トレンドの可視化 / 月次KPIレポートの自動更新
    🎙️ 音声・マルチモーダル系
    13. ElevenLabs / Whisper 音声AI
    ElevenLabsはテキストを自然な音声に変換するAIサービス。WhisperはOpenAIの音声・動画文字起こしAI（無料）。会議録音の議事録作成やナレーション自動生成に使う。
    Whisper：無料 / ElevenLabs：無料（1万文字/月） ElevenLabs Starter 月$5（3万文字）
    東光印刷での活用例：萬財塾・商談の自動議事録 / 動画のAIナレーション音声生成 / 多言語対応の音声ガイド作成
    🎬 映像・3D系
    14. CapCut / Runway 動画編集・AI生成
    CapCutはTikTok（ByteDance）の無料動画編集アプリ。AI自動字幕・BGM・エフェクト機能が充実。RunwayはテキストやAI画像から動画を生成できるAIツール。
    CapCut：無料 / Runway：無料（125クレジット/月） Runway Standard 月$12（625クレジット）
    東光印刷での活用例：商品・店舗紹介のリール動画 / 店頭サイネージ用映像素材 / 印刷物QRコードから動画コンテンツへ
